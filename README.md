# Contador Neubauer

El conteo celular mediante la cámara de Neubauer es una técnica ampliamente utilizada en laboratorio para determinar la concentración de células en una muestra líquida. Este método se basa en la observación microscópica de células contenidas en una cámara con volumen conocido, lo que permite obtener el número de células a una unidad de volumen, generalmente células por mililitro (células/mL).

La cámara de Neubauer está dividida en una cuadrícula con dimensiones definidas. Cada cuadrante tiene un área y profundidad conocida, lo que permite calcular el volumen exacto en el que se encuentran las células. 

El procedimiento consta de: 
1. Primero: una preparación de la muestra donde se mezcla frecuentemente una muestra celular con un colorante de azul tripano. Esto permite diferencias las células viables de las no viables. 

2. Segundo: Calcular el factor de dilución. Al mezclar la muestra con colorante, se genera una dilución. Este factor de dilución se calcula dividiendo el volumen total / volumen de muestra celular. 

3. Tercero: Cargar la cámara. Se coloca una pequeña cantidad de la mezcla en la cámara de Neubauer. 

4. Cuarto: Conteo en cuadrantes. Bajo el microscopio, se cuentan las céulas en varios cuadrantes, esto permite obtener una muestra representativa. 

5. Quinto: Calcular el promedio de células por cuadrantes. 

6. Sexto: Cálculo de concentración. Finalmente, se utiliza la fórmula: 

células/mL = promedio × factor de dilución × 10⁴

El factor 10⁴ proviene de las dimensiones de la cámara y permite convertir el conteo a volumen estándar.

# Relación con la aplicación 

La aplicación desarrollada digitaliza este proceso manual, guiando al ususario paso a paso y reduciendo errores en los cálculos. 


# Flujo de la aplicación

1. Pantalla de dilución
   - El usuario ingresa µL de células y azul tripano
   - Se calcula el factor de dilución

2. Pantalla de conteo
   - Se cuentan células por cuadrante
   - Se almacenan los conteos

3. Pantalla de resultados
   - Se calcula el promedio
   - Se obtiene células/mL con la fórmula:

   células/mL = promedio × factor de dilución × 10⁴
